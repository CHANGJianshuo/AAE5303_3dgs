## 三、参数详解

### `-n` 迭代数

最关键的参数。baseline 只跑了 300 步（推荐的 1%），密度控制根本没启动（warmup=500 > 训练步数 300）。

| 迭代数 | 发生了什么 |
|--------|-----------|
| 0-500 | Warmup 阶段：只优化高斯参数（位置/颜色/透明度/尺寸），不做分裂/克隆 |
| 500-3000 | 密度控制启动：每 100 步分裂/克隆/修剪高斯，点数开始增长 |
| 3000+ | 分辨率提升到原始分辨率，继续优化细节 |
| 3000-30000 | 持续优化 + 密度控制，loss 逐渐收敛 |
| >20000 | 收益递减，loss 变化很小 |

### `-d` 降采样因子

| 值 | 训练分辨率 | 显存 | 速度 | 质量 |
|----|-----------|------|------|------|
| 1 | 2448x2048 (原始) | ~20 GB | 基准 | 最高 |
| 2 | 1224x1024 | ~8 GB | ~4x 快 | 中等 |
| 4 | 612x512 | ~3 GB | ~16x 快 | 较低 |

A100 40GB 用 `-d 1` 完全没问题。

### `--sh-degree` 球谐阶数

| 阶数 | 颜色系数数 | 效果 |
|------|-----------|------|
| 0 | 3 (RGB) | 固定颜色，无视角依赖 |
| 1 | 12 | 简单的视角依赖颜色 |
| 2 | 27 | 中等视角依赖 |
| 3 | 48 | 最精细的视角依赖颜色（高光、反射） |

本数据集几乎全是正射视角，视角变化不大，sh-degree=3 的优势不如多视角场景明显，但 A100 内存够用，不必降低。

### `--densify-grad-thresh` 密度控制梯度阈值

控制何时增加新高斯：
- **0.0002**（默认）：标准密度，平衡质量与速度
- **0.0001**（极致）：更积极地分裂，高斯数量更多，细节更好，但更慢
- **0.0004**（保守）：减少分裂，控制高斯增长，适合内存紧张时

### `--ssim-weight` SSIM 权重

```
Loss = (1 - w) * L1 + w * (1 - SSIM)
```

| 值 | 效果 |
|----|------|
| 0.0 | 纯 L1，像素级精确但可能有块状伪影 |
| 0.2 (默认) | 平衡锐度和结构相似度 |
| 0.4 | 更重视结构，更平滑，适合有噪声的数据 |

## 四、训练命令

### 快速测试

```bash
./opensplat /path/to/AMtown02_colmap \
    -n 7000 \
    -d 2 \
    --sh-degree 2 \
    --ssim-weight 0.2 \
    --refine-every 100 \
    --warmup-length 500 \
    -o amtown02_quick_test.ply \
    2>&1 | tee train_quick.log
```

### 标准训练（推荐）

```bash
./opensplat /path/to/AMtown02_colmap \
    -n 30000 \
    -d 1 \
    --sh-degree 3 \
    --sh-degree-interval 1000 \
    --ssim-weight 0.2 \
    --refine-every 100 \
    --warmup-length 500 \
    --densify-grad-thresh 0.0002 \
    --densify-size-thresh 0.01 \
    --reset-alpha-every 30 \
    --num-downscales 2 \
    --resolution-schedule 3000 \
    --save-every 5000 \
    --val \
    -o amtown02_30k.ply \
    2>&1 | tee train_30k.log
```

### 极致质量

```bash
./opensplat /path/to/AMtown02_colmap \
    -n 30000 \
    -d 1 \
    --sh-degree 3 \
    --sh-degree-interval 1000 \
    --ssim-weight 0.2 \
    --refine-every 100 \
    --warmup-length 500 \
    --densify-grad-thresh 0.0001 \
    --densify-size-thresh 0.005 \
    --reset-alpha-every 30 \
    --num-downscales 2 \
    --resolution-schedule 3000 \
    --save-every 5000 \
    --val \
    -o amtown02_30k_hq.ply \
    2>&1 | tee train_30k_hq.log
```

## 五、预期训练效果

### 与 Baseline 对比

| 指标 | Baseline (CPU, 300步, d=4) | A100 标准 (30k步, d=1) | A100 极致 (30k步, d=1) |
|------|---------------------------|----------------------|----------------------|
| 训练时间 | 25 min | 30-60 min | 60-90 min |
| Initial Loss | 0.2164 | ~0.21 | ~0.21 |
| Final Loss | 0.0888 | **0.01-0.03** | **0.005-0.02** |
| Min Loss | 0.0454 | **< 0.01** | **< 0.005** |
| Loss Reduction | 58.9% | **> 95%** | **> 97%** |
| PSNR | ~20-22 dB | **25-30 dB** | **27-32 dB** |
| SSIM | ~0.75-0.80 | **0.85-0.92** | **0.88-0.94** |
| 高斯数量 | 8.3M (不变) | 10-20M (增长) | 15-25M (更多) |
| 输出 PLY | 2.0 GB | 2-5 GB | 3-6 GB |
| 显存峰值 | N/A | 15-25 GB | 20-30 GB |
